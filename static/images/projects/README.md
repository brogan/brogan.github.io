# Project Images

This folder contains thumbnail images for your projects.

## Guidelines

- **Aspect Ratio**: Use 16:9 ratio (e.g., 800x450px)
- **File Size**: Keep under 500KB for fast loading
- **Format**: PNG or JPG
- **Naming**: Use descriptive names matching your project slugs

## Adding Images

1. Create or export your project screenshot/thumbnail
2. Resize to recommended dimensions
3. Optimize file size (use tools like TinyPNG)
4. Save to this folder
5. Reference in project frontmatter: `thumbnail: "/images/projects/your-image.png"`
